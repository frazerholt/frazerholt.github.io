Poetry Collection

My voice is composed on Microsoft Excel using a structured system. This methodology governs the technical elements, including formulas for counting syllables, establishing meter, and defining the rhyme scheme. Rhymes, internal-rhymes and near-rhymes are visually co-ordinated using spreadsheet color coding. I call this methodolgy 'Stained Glass Poetry' for the grid framework and colour effect as pictured in each poem. No A.I.
